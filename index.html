Способность работать с разными типами данных: текст, изображения, видео, аудио. Есть у ChatGPT-5 и Gemini 2.0.

Diffusion model

Диффузионная модель — основа генераторов изображений (Kandinsky, Midjourney, Stable Diffusion).

Fine-tuning

Донастройка модели на собственных данных для решения конкретных задач.

Prompt engineering

Искусство составления правильных запросов для получения лучших результатов от нейросети.

RAG (Retrieval-Augmented Generation)

Технология, при которой модель ищет информацию в указанных источниках перед генерацией ответа.

AI агент

Автономная программа, выполняющая сложные задачи с помощью нейросетей.

Токенизация

Разбиение текста на мелкие единицы (токены) для обработки нейросетью.
